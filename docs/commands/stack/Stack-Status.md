# **EasyEngine** **(ee)**
### EE Stack Status
Display important status about EasyEngine services.
```bash
ee stack status
# Deprecated way
ee system status
```
Sample Output : 
```bash
^_^[root@example.com:~]# ee stack status

 Memory usage:	74%		     Swap usage:		32.39%

 Nginx:		Running

 MySQL:		Running
 Postfix:	Running
```
